## Getting Started

### Prerequisites
- Python 3.7+
- PyTorch 1.10+
- Additional requirements are listed in `requirements.txt`

### Installation
```bash
git clone https://github.com/niconielsen32/pytorch-tutorials.git
cd pytorch-tutorials
pip install -r requirements.txt
```
